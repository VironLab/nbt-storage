### The minecraft nbt type storage 

## Maven implementation

```xml

<repository>
    <id>vironlab-repo</id>
	<url>https://repo.vironlab.eu/repository/snapshot/</url>
</repository>

<dependency>
	<groupId>eu.vironlab.nbt</groupId>
	<artifactId>nbt-storage</artifactId>
	<version>1.0.0-SNAPSHOT</version>
	<scope>compile</scope>
</dependency>

```
